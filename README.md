# ☕ Lounge Café

A modern and elegant coffee shop web application that delivers a premium café experience online. Lounge Café combines beautiful UI design, responsive layouts, smooth interactions, and a seamless shopping experience for coffee enthusiasts.

- Modern coffee-inspired design

- Clean and elegant interface

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/lounge-cafe.git
cd lounge-cafe
```

### Install Dependencies

```bash
npm install
```

### Environment Variables

Create a `.env` file in the root directory:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
